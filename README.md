# Bound-Runaway

このゲームは、砲台から発射される弾から逃げ回り、長く生き残ることを目指すゲームです。
2.ゲームシステム
砲台からはプレイヤーを追従する弾が放たれ、プレイヤーは弾をできる限り避ける必要があります。発射直前にはインディケーターが見えるようになるため、当たらないように気をつけましょう。
弾の初速は素早いですが、一度の着弾で大幅に減速します。しかし、しばらく経つと再びどんどん速くなっていきます。
現在、同じステージのみで遊ぶことが可能で、ステージセレクト画面から、敵よりも長く生き残る「Stage」と自分一人でどれだけ長く生き残れるかのタイムアタックである「Endless」があります。選択したモードによってゲームシステムに細かい差異があります。
「Stage」モードゲームシステム
ステージ中央のプレイヤーを追尾する砲台から放たれる弾に当たるとゲームオーバーです。
全ての敵が弾に当たるよりも長く生き残ることができればゲームクリアです。
20秒毎にアイテム（通称：電気ドーナツ）が生成され、取ると敵に当てると10秒間敵の動きを10秒止めるフィールドを展開します。

「Endless」ゲームシステム
ステージ中央のプレイヤーを追尾する砲台から放たれる弾に当たるとゲーム終了です。
操作方法
方向キー（←↓↑→）、またはWASDキー：移動
